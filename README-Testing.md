This is Testing Version of ZPM.


HOW TO INSTALL
(start from ~ directory)

sudo apt update && sudo apt upgrade -y && sudo rm -rf /ztmp && sudo apt install git -y && sudo mkdir -p /ztmp && sudo git clone --depth 1 --branch 'Testing' https://github.com/Ignacyyy/Zielina_Package_Manager.git /ztmp && cd /ztmp && sudo chmod +x INSTALL.sh && clear && sudo ./INSTALL.sh && sleep 2 && cd ~
